# Helm Charts

This module makes use of following charts:
- [`cert-manager`](https://github.com/jetstack/cert-manager).
- [`prometheus`](https://github.com/prometheus-operator/prometheus-operator).
- [`prometheus-node-exporter`](https://github.com/prometheus-community/helm-charts/tree/main/charts/prometheus-node-exporter).
- [`alertmanager`](https://github.com/prometheus-community/helm-charts/tree/main/charts/alertmanager).
- [`kube-state-metrics`](https://github.com/prometheus-community/helm-charts/tree/main/charts/kube-state-metrics).
- [`grafane`](https://github.com/grafana/helm-charts/tree/main/charts/grafana).
